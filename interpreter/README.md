# 解释器模式

## 0x01.定义与类型

- 定义：给定一个语言，定义它的文法的一种表示，并定义一个解释器，这个解释器使用该表示来解释语言中的句子。
- 为了解释一种语言，而为语言创建的解释器。
- 类型：行为型

## 0x02.适用场景

- 在某个特定类型问题发生频率足够高 

## 0x03.优点

- 语法由很多类表示，容易改变及扩展此“语言”

## 0x04.缺点

- 当语法规则数目太多时，增加了系统复杂度

## 0x05.相关设计模式

- 解释器模式和适配器模式
    - 适配器模式不需要先知道适配的规则
    - 解释器模式要预先知道语法规则
